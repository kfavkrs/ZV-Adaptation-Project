# Sort Downloads → Photos Videos Slides

Quick instructions for sorting image files out of `~/Downloads` into the Dropbox `Photos Videos Slides` folder.

## Folders involved

- **Source:** `~/Downloads`
- **Destination:** `~/Seattle Unity Dropbox/Seattle Unity Team Folder/Photos Videos Slides/`

## Sorting rules

| File pattern | Destination subfolder |
|---|---|
| Image with a date in the filename (event slide image) | `\|\|\| Dated Events/` |
| Image of the building | `Building/` |
| Image used for an annually recurring event | `\|\|\| Recurring Events/Annual Recurring/` |
| `.pptx` (PowerPoint) or `.mp4` (video) | `\|\|\| Slides/` |
| Image file (.jpg/.png) — even if "Slide" is in the name | **NOT** `\|\|\| Slides/` |

**Key rule:** `||| Slides/` is for PowerPoints and videos only. The word "Slides" refers to file format, not visual shape. Image files always go in a category folder (Dated Events, Recurring Events, Building, People, etc.).

## Files that do NOT belong here

PDFs, spreadsheets, audio recordings, and other non-image files don't go in `Photos Videos Slides` — leave them for a separate decision.

## Process

1. List loose files in Downloads.
2. Propose a destination for each one in a table.
3. Confirm anything ambiguous before moving.
4. For each file: copy to destination, then delete the original from Downloads.
