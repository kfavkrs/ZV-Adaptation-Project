# Pitch Page — Investor/Collaborator Explainer

The investor/collaborator-facing explainer for the Adaptation Blueprint /
Alien Anthropologist Mission Program, built with Claude Design.

- **`prompt.md`** — the final prompt used to generate the page
- **`design-rationale.md`** — how the brief evolved across the drafting conversation
  (audience/scope/aesthetic decisions, and the three iterations that shaped the final prompt),
  plus a fourth section documenting what the actual Claude Design build added beyond the
  prompt — an interactive prioritization-engine demo, named multi-mode debrief capture, an
  explicit mission-architecture design space, party-size as a scaling parameter, and a new
  consented co-op network feature that upgrades one of the Octalysis drivers
- **`index.html`** — the generated page itself. Open directly in a browser, or serve via
  GitHub Pages for a shareable link. Note: it currently has one inconsistency (origin city
  reads "Seattle" in the hero and "Austin" in the footer) worth fixing before sharing.
